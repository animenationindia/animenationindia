## 🚀 About The Project

**Anime Nation India** is a premium, high-performance anime database platform built with Next.js App Router. It offers a stunning "Deep Space Cinematic" dark UI with glassmorphism effects and neon accents. Users can discover trending anime, check live release schedules, watch official trailers, and explore extensive details about their favorite characters and voice actors.

## ✨ Key Features

- 🎥 **Cinematic UI/UX:** A stunning dark mode interface (`#05050A`) with vibrant Purple (`#8B5CF6`) and Cyan (`#06B6D4`) neon glows, featuring a customized Hero auto-slider and smooth Swiper.js carousels.
- 📊 **Comprehensive Anime Database:** Built on top of the powerful **AniList GraphQL API**, featuring Trending, Upcoming, Top Rated, and Recently Updated anime lists.
- 🦸‍♂️ **In-Depth Character & Staff Profiles:** Integrated with the **Jikan (MyAnimeList) API** to fetch extensive biographies, Animeography grids, and Dual-Look Voice Acting Role cards.
- ⏱️ **Live Release Schedule:** A dedicated schedule page featuring countdown timers for the week's upcoming anime episodes.
- 🎬 **Official Trailers:** An interactive grid and live-search dropdown for official YouTube anime trailers.
- 📰 **Anime News & Announcements:** A dedicated news portal with custom categories and a featured announcement banner.
- 🔍 **Real-Time Search:** Instant global search functionality integrated directly into the Navbar.
- 📱 **Fully Responsive:** Meticulously designed to look and work perfectly across desktop, tablet, and mobile devices without horizontal overflow bugs.
- 🤖 **SEO Optimized:** Dynamically generated `sitemap.xml`, `robots.txt`, and OpenGraph metadata for enhanced Google discoverability.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router, Server Components)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Carousels:** [Swiper.js](https://swiperjs.com/)
- **Data Fetching:** Native `fetch` with Next.js caching & revalidation
- **APIs:** [AniList GraphQL](https://anilist.gitbook.io/) & [Jikan REST (MAL)](https://docs.api.jikan.moe/)

## ⚙️ Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have Node.js installed (v18.x or later).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/animenationindia.git
   ```
2. Navigate to the project directory:
   ```bash
   cd animenationindia
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:3000`
